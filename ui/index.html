🔧 Quick Start Guide

1. Configure APIs

Go to Settings and configure:

  • LiveKit Cloud (URL, API Key, Secret)
  • Google Gemini (API Key, Model, Voice)
  • Vobiz SIP (Domain, Username, Password, Outbound Number)
  • Supabase (URL, Service Key)

2. Create SIP Trunk

In Settings → LiveKit, click "⚡ Create SIP Trunk" after configuring Vobiz credentials.

3. Test Single Call

Go to Single Call tab and test with your own number.

4. Create Agent Profile

Go to Agents tab and create a profile with your business details.
